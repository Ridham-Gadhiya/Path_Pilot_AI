# PathPilot AI

PathPilot AI is an AI-powered study and career recommendation assistant designed to help students and job seekers discover the best next learning content based on their goals, interests, and learning journey.

## Project Goal
To recommend useful study videos and courses instead of making users endlessly search for what to learn next.

## Current Status
Project setup completed. Building dataset pipeline and recommendation engine next.

## Planned Features
- Goal-based recommendations
- Topic-based learning suggestions
- Learning journey progression logic
- Content-based recommendation engine
- Hybrid ranking system
- Streamlit web app
